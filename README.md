# 診察用 体調・既往歴まとめシート

診察時に、症状・既往歴・服薬情報・検査結果を整理して医師に伝えやすくするためのWebアプリです。

## 主な機能

- 受診科の選択
- 症状の選択・自由入力
- 症状経過の記録
- 既往歴の整理
- 服薬情報の整理
- 検査・診察結果の記録
- 医師に相談したいことの整理
- 印刷用シート作成
- AI要約用テキスト作成

## 使い方

1. 各項目を入力または選択します
2. 右側または下部のプレビューで内容を確認します
3. 必要に応じて印刷します
4. AI要約用テキストをコピーして、自分のChatGPTに貼り付けることもできます

## 注意事項

このアプリは診察時の説明補助を目的としたもので、診断を行うものではありません。

強い胸痛、息苦しさ、麻痺、ろれつが回らない、激しい頭痛、高度な血圧上昇などがある場合は、すぐに医療機関へ相談してください。
